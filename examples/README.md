# RunGuard Examples

This folder contains public-safe workflow examples for Marketplace users.

- `workflows/risky-review.yml`: intentionally risky workflow that demonstrates RunGuard findings.
- `workflows/safer-review.yml`: safer baseline pattern for the same general workflow.
- `sample-report.md`: generated RunGuard Markdown output from the risky workflow.

Use these examples in demos, launch posts, Marketplace screenshots, and customer discovery.
